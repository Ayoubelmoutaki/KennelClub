# Marrakech Dog Hotel (Fixed)

This version adds `@vitejs/plugin-react` and an ErrorBoundary to help avoid blank pages in production.

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Vercel Settings
- Framework Preset: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`
